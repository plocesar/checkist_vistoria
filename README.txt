# Check-list de Vistoria

Este pacote contém o arquivo HTML do checklist de vistoria interativo.

## Como usar localmente
1. Extraia o arquivo ZIP.
2. Abra o arquivo `checklist-vistoria.html` em qualquer navegador (Chrome, Edge, Firefox).
3. Preencha os campos e marque os checkboxes diretamente no navegador.

## Como publicar online (Netlify Drop)
1. Acesse: https://app.netlify.com/drop
2. Arraste o arquivo `checklist-vistoria.html` ou o ZIP inteiro para a tela do Netlify Drop.
3. Aguarde alguns segundos, e você receberá um link público no formato:
   https://nomesitealeatorio.netlify.app

## Dicas
- Você pode personalizar o nome do site dentro do painel do Netlify (Site Settings → Change site name).
- Sempre que quiser atualizar, basta arrastar novamente a nova versão do arquivo.
